# Blue Ghost Lab

[![Website](https://img.shields.io/badge/WEBSITE-7DD3FC?style=for-the-badge&logoColor=white)](https://blueghostlab.com)

We build on **Reticulum** to support the people already using it, and to make space for the ones who haven't found it yet. Privacy isn't a feature we add, it's the floor we start from.

# What we believe

- **Local first.** A network you can't run yourself isn't really yours. The ones worth building work without permission and without anyone's servers in the middle.
- **Community:** Tools should be accessible and belong to the people who use them. They should be built where people can see them.
- **Resilience:** Whatever we build should work with what the rest of the Reticulum community builds. The ecosystem is whats important to us.

# What we do

Reticulum is powerful. It's also not accessible yet for Grandma. We use what we know - UX, design, software, docs - to make Reticulum more approachable, so more communities can run decentralized comms on their own terms.

If you're building on RNS, trying to, or just figuring out what it is, reach out if your curious to learn more about what we're doing.
